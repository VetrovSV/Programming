## Blocks ‚Äî –¥–ª—è –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã—Ö —Å—Ü–µ–Ω–∞—Ä–∏–µ–≤

–ï—Å–ª–∏ –Ω—É–∂–Ω–æ —Å–¥–µ–ª–∞—Ç—å –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å –∏–∑ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤, —É–ø—Ä–∞–≤–ª—è—Ç—å –º–∞–∫–µ—Ç–æ–º, —Å—Ç—Ä–æ–∏—Ç—å –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –¥–µ–π—Å—Ç–≤–∏–π (–Ω–∞–ø—Ä–∏–º–µ—Ä, –∑–∞–≥—Ä—É–∑–∫–∞ —Ñ–∞–π–ª–∞ ‚Üí –≤—ã–±–æ—Ä –ø–∞—Ä–∞–º–µ—Ç—Ä–∞ ‚Üí –æ–±—Ä–∞–±–æ—Ç–∫–∞), —Å—Ç–æ–∏—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å Gradio Blocks:
- **Row, Column, Tab** ‚Äî –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã –¥–ª—è –æ—Ä–≥–∞–Ω–∏–∑–∞—Ü–∏–∏ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞.

**–ü—Ä–∏–º–µ—Ä: –¥–≤–µ –≤–∫–ª–∞–¥–∫–∏ ‚Äî –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ –∏ –≥—Ä–∞—Ñ–∏–∫**
```python
import gradio as gr
from sklearn.linear_model import LinearRegression
import numpy as np
import matplotlib.pyplot as plt

model = LinearRegression()
X = np.array([[1],[2],[3],[4]])
Y = np.array([2,4,6,8])
model.fit(X,Y)

def predict(x):
    return float(model.predict([[x]]))

def plot():
    fig, ax = plt.subplots()
    ax.scatter(X, Y)
    ax.plot(X, model.predict(X), color="red")
    return fig

with gr.Blocks() as demo:
    with gr.Tab("–ü—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ"):
        inp = gr.Number(label="X")
        out = gr.Number(label="Y")
        btn = gr.Button("–°–¥–µ–ª–∞—Ç—å –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ")
        btn.click(predict, inp, out)
    with gr.Tab("–ì—Ä–∞—Ñ–∏–∫"):
        plot_btn = gr.Button("–ü–æ—Å—Ç—Ä–æ–∏—Ç—å –≥—Ä–∞—Ñ–∏–∫")
        plot_area = gr.Plot()
        plot_btn.click(plot, None, plot_area)

demo.launch()
```
